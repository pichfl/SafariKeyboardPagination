KeyboardPagination

Safari extension for adding keyboard support to websites with pagination.

View on GitHub Download .zip Download .tar.gz

This extension searched for pagination elements in a page and tries to make them accessible via the keyboard cursor keys.

Currently, only page number based paginations are supported, a more extensive solution which also looks for previous/next page links might be added later.
